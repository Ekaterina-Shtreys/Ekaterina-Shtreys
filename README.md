# Екатерина Штрейс — аналитик данных

Я аналитик данных с проектным опытом работы в Python, SQL, Excel и Yandex DataLens.

## Что я умею

* проводить предобработку и исследовательский анализ данных;
* писать SQL-запросы с фильтрацией, агрегацией, JOIN, подзапросами, CTE и оконными функциями;
* рассчитывать продуктовые и бизнес-метрики;
* проверять статистические гипотезы и анализировать результаты A/B-тестов;
* создавать графики, отчёты и интерактивные дашборды;
* формулировать выводы и рекомендации для принятия решений.
